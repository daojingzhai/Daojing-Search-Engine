                <h1 id="一对多的操做">一对多的操做</h1>

<p>开发过程中表示一对多关系，一般称一方为为主表或一表，多方为从表或者多表，为了表示一对多的关系，一般会在多表的一方添加一个字段（主表名称_id）字段类型一般和主表的主键类型一致，并称这个字段为外键； <br>
例如订单和用户的一对多关系中我们使用如下代码表示</p>



<pre class="prettyprint"><code class="language-sql hljs "><span class="hljs-comment">--创建用户表</span>
<span class="hljs-operator"><span class="hljs-keyword">create</span> <span class="hljs-keyword">table</span> <span class="hljs-keyword">user</span>(
id <span class="hljs-keyword">int</span> <span class="hljs-keyword">primary</span> <span class="hljs-keyword">key</span> auto_increment,
username <span class="hljs-keyword">varchar</span>(<span class="hljs-number">20</span>)
);</span></code></pre>

<pre class="prettyprint"><code class="language-sql hljs "><span class="hljs-comment">--创建订单表</span>
<span class="hljs-operator"><span class="hljs-keyword">create</span> <span class="hljs-keyword">table</span> orders(
id <span class="hljs-keyword">int</span> <span class="hljs-keyword">primary</span> <span class="hljs-keyword">key</span> auto_increment,
price <span class="hljs-keyword">double</span>,
user_id <span class="hljs-keyword">int</span>
);</span></code></pre>

<p>为了保证数据的完整性和有效性，添加约束（外键约束） <br>
    在多表一方添加外键约束 <br>
    格式 <code>alter table 多表名称 add foreign key(外键名称) references 表名称(主键)</code> <br>
    例如</p>

<pre class="prettyprint"><code class="language-sql hljs "><span class="hljs-operator"><span class="hljs-keyword">alter</span> <span class="hljs-keyword">table</span> orders <span class="hljs-keyword">add</span> <span class="hljs-keyword">foreign</span> <span class="hljs-keyword">key</span>(user_id) <span class="hljs-keyword">references</span> <span class="hljs-keyword">user</span>(id);</span></code></pre>

<p>添加了外键约束有如下特点:</p>

<ul>
<li><p>主表中不能删除从表中已引用的数据</p></li>
<li><p>从表中不能添加主表没有的数据</p></li>
</ul>

<h1 id="多对多的操做">多对多的操做</h1>

<p>开发过程中表示多对多关系，一般引入一张中间表，在中间表中存放两张表的主键，一般还会将这两个主键设为中间表的联合主键 <br>
例如订单和商品的多对多关系中我们使用如下代码表示</p>

<pre class="prettyprint"><code class="language-sql hljs "><span class="hljs-comment">--创建订单表</span>
<span class="hljs-operator"><span class="hljs-keyword">create</span> <span class="hljs-keyword">table</span> orders(
id <span class="hljs-keyword">int</span> <span class="hljs-keyword">primary</span> <span class="hljs-keyword">key</span> auto_increment,
price <span class="hljs-keyword">double</span>,
user_id <span class="hljs-keyword">int</span>
);</span></code></pre>



<pre class="prettyprint"><code class="language-sql hljs "><span class="hljs-comment">--创建商品表</span>
<span class="hljs-operator"><span class="hljs-keyword">create</span> <span class="hljs-keyword">table</span> product(
id <span class="hljs-keyword">int</span> <span class="hljs-keyword">primary</span> <span class="hljs-keyword">key</span> auto_increment,
name <span class="hljs-keyword">varchar</span>(<span class="hljs-number">20</span>),
price <span class="hljs-keyword">double</span>
);</span></code></pre>

<pre class="prettyprint"><code class="language-sql hljs "><span class="hljs-comment">--创建中间表</span>
<span class="hljs-operator"><span class="hljs-keyword">create</span> <span class="hljs-keyword">table</span> oder_product(
product_id <span class="hljs-keyword">int</span>,
orders_id <span class="hljs-keyword">int</span>
);</span></code></pre>

<p>为了保证数据的完整性和有效性，添加约束（外键约束） <br>
多对多的关系在引入中间表后就可以看成是两个一对多，所以在中间表中添加两个外键约束，外键约束如下</p>

<pre class="prettyprint"><code class="language-sql hljs "><span class="hljs-comment">--添加外键约束</span>
<span class="hljs-operator"><span class="hljs-keyword">alter</span> <span class="hljs-keyword">table</span> oder_product <span class="hljs-keyword">add</span> <span class="hljs-keyword">foreign</span> <span class="hljs-keyword">key</span>(product_id) <span class="hljs-keyword">references</span> product(id);</span>
<span class="hljs-operator"><span class="hljs-keyword">alter</span> <span class="hljs-keyword">table</span> oder_product <span class="hljs-keyword">add</span> <span class="hljs-keyword">foreign</span> <span class="hljs-keyword">key</span>(orders_id) <span class="hljs-keyword">references</span> orders(id);</span></code></pre>



<h1 id="多表查询">多表查询</h1>



<h2 id="笛卡儿积">笛卡儿积</h2>

<p>多张表无条件的联合查询，没有任何意义</p>

<pre class="prettyprint"><code class="language-sql hljs "><span class="hljs-operator"><span class="hljs-keyword">select</span> a.*,b.* <span class="hljs-keyword">from</span> a,b;</span></code></pre>

<h2 id="内连接">内连接</h2>

<p>语法格式一：显试的内连接</p>



<pre class="prettyprint"><code class="language-sql hljs "><span class="hljs-operator"><span class="hljs-keyword">select</span> a.*,b.* <span class="hljs-keyword">from</span> a [<span class="hljs-keyword">inner</span>] <span class="hljs-keyword">join</span> b <span class="hljs-keyword">on</span> ab的连接条件</span></code></pre>

<p>语法格式二：隐试的内连接</p>



<pre class="prettyprint"><code class="language-sql hljs "><span class="hljs-operator"><span class="hljs-keyword">select</span> a.*,b.* <span class="hljs-keyword">from</span> a,b <span class="hljs-keyword">where</span> ab的连接条件</span></code></pre>



<h2 id="外连接">外连接</h2>

<p>左外连接:</p>



<pre class="prettyprint"><code class="language-sql hljs "><span class="hljs-operator"><span class="hljs-keyword">select</span> a.*,b.* <span class="hljs-keyword">from</span> a <span class="hljs-keyword">left</span> [<span class="hljs-keyword">outer</span>] <span class="hljs-keyword">join</span> b <span class="hljs-keyword">on</span> 连接条件</span></code></pre>

<p>含义</p>

<pre><code>先展示join左边表(a)的数据，根据条件再查询join右边表(b)的数据，满足条件则展示，不满足条件以null展示。
</code></pre>

<p>右外连接</p>

<pre class="prettyprint"><code class="language-sql hljs "><span class="hljs-operator"><span class="hljs-keyword">select</span> a.*,b.* <span class="hljs-keyword">from</span> a <span class="hljs-keyword">right</span> [<span class="hljs-keyword">outer</span>] <span class="hljs-keyword">join</span> b <span class="hljs-keyword">on</span> 连接条件</span></code></pre>

<pre><code>含义:
先展示join右边表(b)的数据，根据条件再查询join左边表(a)的数据，满足条件则展示，不满足条件以null展示。
</code></pre>            </div>
