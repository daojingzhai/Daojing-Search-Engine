<p>新建学生表：student</p>
<p></p><pre><code class="language-sql">CREATE TABLE `student` (
  `id` varchar(10) NOT NULL,
  `name` varchar(10) NOT NULL,
  `sex` varchar(10) NOT NULL,
  `dept` varchar(10) DEFAULT NULL,
  PRIMARY KEY (`id`)

<p>新建教师表：teacher<br /></p>
<p></p><pre><code class="language-sql"> create table teacher
 (
 Tid varchar(10) not null primary key,
 Tname varchar(10) not null,
 Title varchar(10) not null,
 dept varchar(10));</code></pre>
<p><img src="https://img-blog.csdn.net/20170731213310989?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMjE4MDg5NjE=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" alt="" /><br />
新建课程表：course</p>
<p>课程表中用教师的工号作为外键，要先建立教师表，不能先建立课程表<br /></p>
<p></p><pre><code class="language-sql"> create table course
 (
 Cid varchar(10) not null primary key,
 Cname varchar(10) not null,
 Tid varchar(10),
 foreign key(Tid)
 references teacher(Tid));</code></pre><img src="https://img-blog.csdn.net/20170731213335284?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMjE4MDg5NjE=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" alt="" /><br /><p>新建选课表：SC</p>
<p></p><pre><code class="language-sql">create table SC
(
Sid varchar(10) not null,
Cid varchar(10) not null,
score int,
primary key(Sid,Cid),
foreign key(Sid) references student(id),
foreign key(Cid) references course(Cid));</code></pre><img src="https://img-blog.csdn.net/20170731213355594?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMjE4MDg5NjE=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" alt="" /><br />
然后插入数据，插入数据是要先插入主表teacher,再插入副表course,先插入主表student,course,再插入副表SC

<p><img src="https://img-blog.csdn.net/20170731213841729?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMjE4MDg5NjE=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" alt="" /><br /></p>

<p><img src="https://img-blog.csdn.net/20170731213929901?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMjE4MDg5NjE=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" alt="" /><br /></p>
<p>选课表中的信息：</p>
<p><img src="https://img-blog.csdn.net/20170731214029275?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMjE4MDg5NjE=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" alt="" /></p>
<p>新建一个存贮过程setScore()来设置成绩表SC中的成绩:</p>
<p></p><pre><code class="language-sql"> delimiter &amp;&amp;
 create procedure setScore(in studentid varchar(10),in courseid varchar(10),in core int)
 begin
  update SC set score=core where Sid=studentid and Cid=courseid;
 end&amp;&amp;</code></pre><br />
通过调用存储过程来跟新成绩表SC中学生的成绩：
<p></p><pre><code class="language-sql"> call setScore("B1000","C002",99);</code></pre>结果：
<p><img src="https://img-blog.csdn.net/20170731214558912?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMjE4MDg5NjE=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center" alt="" /></p>
<p>可以看到学生“B1000”,选的"C002"这门课的成绩已经设置成99了。</p>
<p>可以看到有了存储过程之后就可以不用每次输入重复，而且一长串的SQL语句。</p>
