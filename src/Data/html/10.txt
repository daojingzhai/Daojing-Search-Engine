                <h3 id="1-外键">1. 外键</h3>

<p>外键是指应用另外的表的一列或者多列，被引用的列应该具有主键约束或唯一性约束，外键用于建立和加强两个表之间的联系。 <br>
建立数据库及数据表：</p>



<pre class="prettyprint"><code class=" hljs sql"><span class="hljs-operator"><span class="hljs-keyword">create</span> <span class="hljs-keyword">database</span> bs4;</span>
use db4;
<span class="hljs-operator"><span class="hljs-keyword">create</span> <span class="hljs-keyword">table</span> grade(
id <span class="hljs-keyword">int</span>(<span class="hljs-number">4</span>) <span class="hljs-keyword">not</span> <span class="hljs-keyword">null</span> <span class="hljs-keyword">primary</span> <span class="hljs-keyword">key</span>,
name <span class="hljs-keyword">varchar</span>(<span class="hljs-number">36</span>));</span>
<span class="hljs-operator"><span class="hljs-keyword">create</span> <span class="hljs-keyword">table</span> student(
sid <span class="hljs-keyword">int</span>(<span class="hljs-number">4</span>) <span class="hljs-keyword">not</span> <span class="hljs-keyword">null</span> <span class="hljs-keyword">primary</span> <span class="hljs-keyword">key</span>,
sname <span class="hljs-keyword">varchar</span>(<span class="hljs-number">36</span>),
gid <span class="hljs-keyword">int</span>(<span class="hljs-number">4</span>) <span class="hljs-keyword">not</span> <span class="hljs-keyword">null</span>);</span></code></pre>

<p>此时student的gid字段表示学生的年级非空，即可表示student表的外键，两个表可以通过gid建立联系。 <br>
为student表添加外键：</p>



<pre class="prettyprint"><code class=" hljs sql"><span class="hljs-operator"><span class="hljs-keyword">alter</span> <span class="hljs-keyword">table</span> student <span class="hljs-keyword">add</span> <span class="hljs-keyword">constraint</span> FK_ID <span class="hljs-keyword">foreign</span> <span class="hljs-keyword">key</span>(gid) <span class="hljs-keyword">references</span> grade(id);</span></code></pre>

<p>其中FK_ID是外键名，使用desc查看表的结果：</p>



<pre class="prettyprint"><code class=" hljs asciidoc"><span class="hljs-header">mysql&gt; desc grade;
+-------+-------------+------+-----+---------+-------+</span>
<span class="hljs-header">| Field | Type        | Null | Key | Default | Extra |
+-------+-------------+------+-----+---------+-------+</span>
| id    | int(4)      | NO   | PRI | NULL    |       |
<span class="hljs-header">| name  | varchar(36) | YES  |     | NULL    |       |
+-------+-------------+------+-----+---------+-------+</span>
2 rows in set (0.01 sec)

<span class="hljs-header">mysql&gt; desc student;
+-------+-------------+------+-----+---------+-------+</span>
<span class="hljs-header">| Field | Type        | Null | Key | Default | Extra |
+-------+-------------+------+-----+---------+-------+</span>
| sid   | int(4)      | NO   | PRI | NULL    |       |
| sname | varchar(36) | YES  |     | NULL    |       |
<span class="hljs-header">| gid   | int(4)      | NO   | MUL | NULL    |       |
+-------+-------------+------+-----+---------+-------+</span>
3 rows in set (0.00 sec)</code></pre>

<p>删除外键约束：</p>



<pre class="prettyprint"><code class=" hljs sql"><span class="hljs-operator"><span class="hljs-keyword">alter</span> <span class="hljs-keyword">table</span> student <span class="hljs-keyword">drop</span> <span class="hljs-keyword">foreign</span> <span class="hljs-keyword">key</span> FK_ID;</span></code></pre>



<h3 id="2操作关联表">2.操作关联表</h3>



<h4 id="21添加数据">2.1添加数据</h4>

<p>首先创建外键</p>



<pre class="prettyprint"><code class=" hljs sql"><span class="hljs-operator"><span class="hljs-keyword">alter</span> <span class="hljs-keyword">table</span> student <span class="hljs-keyword">add</span> <span class="hljs-keyword">constraint</span> FK_ID <span class="hljs-keyword">foreign</span> <span class="hljs-keyword">key</span>(gid) <span class="hljs-keyword">references</span> grade(id);</span></code></pre>

<p>此时，student与grade是多对一的关系。要添加数据首先要对主表grade进行添加数据，然后对从表student。<strong>主表：在数据库中建立的表格即Table，其中存在主键(primary key)用于与其它表相关联，并且作为在主表中的唯一性标识。从表：以主表的主键（primary key）值为外键 (Foreign Key)的表，可以通过外键与主表进行关联查询。从表与主表通过外键进行关联查询。</strong> <br>
首先为grade表添加数据：</p>



<pre class="prettyprint"><code class=" hljs sql"><span class="hljs-operator"><span class="hljs-keyword">insert</span> <span class="hljs-keyword">into</span> grade <span class="hljs-keyword">values</span>(<span class="hljs-number">1</span>,<span class="hljs-string">'信管1'</span>),(<span class="hljs-number">2</span>,<span class="hljs-string">'信管2'</span>);</span></code></pre>

<p>在主表班级为1，2的情况下，从表student的gid也只能为1，2，所以插入数据如下：</p>



<pre class="prettyprint"><code class=" hljs sql"><span class="hljs-operator"><span class="hljs-keyword">insert</span> <span class="hljs-keyword">into</span> student(sid,sname,gid) <span class="hljs-keyword">values</span>(<span class="hljs-number">1</span>,<span class="hljs-string">'王'</span>,<span class="hljs-number">1</span>),(<span class="hljs-number">2</span>,<span class="hljs-string">'张'</span>,<span class="hljs-number">2</span>),(<span class="hljs-number">3</span>,<span class="hljs-string">'李'</span>,<span class="hljs-number">1</span>),(<span class="hljs-number">4</span>,<span class="hljs-string">'赵'</span>,<span class="hljs-number">2</span>);</span></code></pre>



<h4 id="22-删除数据">2.2 删除数据</h4>

<p>删除id=1,信管1班数据，只能选择先删除从表（student）中的数据，然后再删除主表数据。 <br>
语句：</p>



<pre class="prettyprint"><code class=" hljs sql"><span class="hljs-operator"><span class="hljs-keyword">delete</span> <span class="hljs-keyword">from</span> student <span class="hljs-keyword">where</span> sname=<span class="hljs-string">'王'</span>；
<span class="hljs-keyword">delete</span> <span class="hljs-keyword">from</span> student <span class="hljs-keyword">where</span> sname=<span class="hljs-string">'李'</span>；</span></code></pre>

<p>然后删除主表grade中的数据：</p>



<pre class="prettyprint"><code class=" hljs sql"><span class="hljs-operator"><span class="hljs-keyword">delete</span> <span class="hljs-keyword">from</span> grade <span class="hljs-keyword">where</span> id=<span class="hljs-number">1</span>;</span></code></pre>

<h4 id="32内链接查询">3.2内链接查询</h4>

<p>常见的查询方式，使用比较运算符对两个表的数据进行比较，只有满足条件的记录才会出现在查询结果中。 <br>
语法<code>select [查询字段] from [表1] join [表2] on [查询条件]</code> <br>
查询employee id和department id相同的数据。</p>

<pre class="prettyprint"><code class=" hljs asciidoc"><span class="hljs-header">select employee.did,employee.name,department.dname,department.did from department join employee on department.did = employee.did;
+-----+------+-----------+-----+</span>
<span class="hljs-header">| did | name | dname     | did |
+-----+------+-----------+-----+</span>
|   1 | 王   | 网络部    |   1 |
|   1 | 张   | 网络部    |   1 |
<span class="hljs-header">|   2 | 李   | 媒体部    |   2 |
+-----+------+-----------+-----+</span>
3 rows in set (0.00 sec)</code></pre>

<p>上述查询使用where语句也可以实现。查询结果一致。</p>

<pre class="prettyprint"><code class=" hljs avrasm">select employee<span class="hljs-preprocessor">.did</span>,employee<span class="hljs-preprocessor">.name</span>,department<span class="hljs-preprocessor">.dname</span>,department<span class="hljs-preprocessor">.did</span> from department,employee where department<span class="hljs-preprocessor">.did</span> = employee<span class="hljs-preprocessor">.did</span><span class="hljs-comment">;</span>
</code></pre>

<p>在查询时可能会涉及两个要查询的表是同一个表，比如，要查询王所在部门有哪些员工，只使用employee表即可完成查询。称之为自然链接查询，是内链接的一种，在物理上是同一个表，逻辑上分为两个表。</p>

<pre class="prettyprint"><code class=" hljs sql"><span class="hljs-operator"><span class="hljs-keyword">select</span> p1.* <span class="hljs-keyword">from</span> employee p1 <span class="hljs-keyword">join</span> employee p2 <span class="hljs-keyword">on</span> p1.did=p2.did <span class="hljs-keyword">where</span> p2.name=<span class="hljs-string">'王'</span>;</span></code></pre>

<p>查询结果如下：</p>



<pre class="prettyprint"><code class=" hljs asciidoc"><span class="hljs-header">select p1.* from employee p1 join employee p2 on p1.did=p2.did where p2.name='王';
+----+------+------+-----+</span>
<span class="hljs-header">| id | name | age  | did |
+----+------+------+-----+</span>
|  1 | 王   |   20 |   1 |
<span class="hljs-header">|  2 | 张   |   22 |   1 |
+----+------+------+-----+</span>
2 rows in set (0.00 sec)</code></pre>

<h3 id="4子查询">4.子查询</h3>

<p>查询结果作为外层查询的条件，通常使用in,exists,any,all操作符。</p>



<h4 id="41-in">4.1 in</h4>

<p>查询年龄存在22岁的部门</p>

<pre class="prettyprint"><code class=" hljs asciidoc"><span class="hljs-header">select * from department where did in(select did from employee where age=22);
+-----+-----------+</span>
<span class="hljs-header">| did | dname     |
+-----+-----------+</span>
<span class="hljs-header">|   1 | 网络部    |
+-----+-----------+</span>
1 row in set (0.01 sec)</code></pre>



<h4 id="42-exists">4.2 exists</h4>

<p>不返回数据，只返回true或者false，当返回true时外层查询才执行。 <br>
本例，内层存在，执行外层。</p>

<pre class="prettyprint"><code class=" hljs asciidoc"><span class="hljs-header">select * from department where exists(select did from employee where age&gt;20);
+-----+-----------+</span>
<span class="hljs-header">| did | dname     |
+-----+-----------+</span>
|   1 | 网络部    |
|   2 | 媒体部    |
|   3 | 研发部    |
<span class="hljs-header">|   5 | 人事部    |
+-----+-----------+</span>
4 rows in set (0.00 sec)</code></pre>



<h4 id="43-any">4.3 any</h4>

<p>只要满足内层子查询中的任意一个比较条件，就返回一个结果作为外层查询条件。 <br>
本例子由于employee存在did为1，所以返回department2，3，5</p>

<pre class="prettyprint"><code class=" hljs asciidoc"><span class="hljs-header">select * from department where did &gt; any(select did from employee);
+-----+-----------+</span>
<span class="hljs-header">| did | dname     |
+-----+-----------+</span>
|   2 | 媒体部    |
|   3 | 研发部    |
<span class="hljs-header">|   5 | 人事部    |
+-----+-----------+</span>
3 rows in set (0.00 sec)</code></pre>



<h4 id="44-all">4.4 all</h4>

<p>与any相似，但是需要满足全部条件。</p>

<pre class="prettyprint"><code class=" hljs asciidoc"><span class="hljs-header">select * from department where did &gt; all(select did from employee);
+-----+-----------+</span>
<span class="hljs-header">| did | dname     |
+-----+-----------+</span>
<span class="hljs-header">|   5 | 人事部    |
+-----+-----------+</span>
1 row in set (0.00 sec)</code></pre>



<h4 id="45-带比较运算符的子查询">4.5 带比较运算符的子查询</h4>

<p>还可以使用：</p>

<pre class="prettyprint"><code class=" hljs asciidoc"><span class="hljs-header">select * from department where did=(select did from employee where name='王');
+-----+-----------+</span>
<span class="hljs-header">| did | dname     |
+-----+-----------+</span>
<span class="hljs-header">|   1 | 网络部    |
+-----+-----------+</span>
1 row in set (0.00 sec)</code></pre>            </div>
